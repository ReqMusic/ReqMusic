# ReqMusic
ReqMusic is a personal project I'm going to undertake which will recommend music to the user based on their current music interests. I plan on utilizing Spotify's APIs to accomplish this and this app is a web application which is built upon Ruby on Rails.

# Development Information
### Starting the Server
In the ReqMusic directory of the project run the following to start the local development server:\
```rails server```

### Ruby & Rails Versions
Versions as of Feb 21, 24\
```ruby 3.2.2```\
```rails 7.1.3.2```
